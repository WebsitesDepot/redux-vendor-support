== Description ==

The Redux Vendor Library contains the ACE Editor and select2 vendor libraries as
the Redux Framework CDN fallback.

Redux Framework uses contains support for the ACE Editor field and select2, and
by default uses a CDN to load vendor support files for those features.  There are
certain instances where a CDN may not work or in not preferable (or permitted).
This library - when installed and activated - automatically provides ACE and 
select2 vendor support for Redux Framework.

== Frequently Asked Questions ==

=Why doesn't think plugin do anything?=

This plugin is a support library for Redux Framework, when the use of a CDN is
unavailable, preferred or prohibited.  It is designed to be used exclusively
with Redux Framework for ACE Editor and select2 support.  Redux Framework users
do not to do anything other than activate this plugin.  Redux Framework does that
rest.
